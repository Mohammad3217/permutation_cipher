# 🔐 Permutation Cipher - رمزنگاری جایگشتی

یک API کامل و رابط کاربری برای رمزنگاری و رمزگشایی با استفاده از روش جایگشت بلوکی (Permutation Cipher)
پروژه درس مبانی رایانش امن دانشگاه  

## 🌐 پیش‌نمایش زنده

https://permutations-cipher-delta.vercel.app/index.html

## ✨ ویژگی‌ها

- **رمزنگاری بلوکی** با طول بلوک دلخواه
- **API با Node.js + Express + TypeScript**
- **اعتبارسنجی کامل ورودی** با Zod
- **رابط کاربری** با Tailwind CSS
- **قابلیت کپی خودکار** نتیجه
- **مدیریت خطا** با توست‌های زیبا
- **Rate Limiting** برای امنیت بیشتر
- **Sanitize ورودی‌ها** برای جلوگیری از حملات
